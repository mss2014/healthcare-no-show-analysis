README: Healthcare No-Show Analysis and Dashboard Project

Project Overview
This project focuses on analyzing healthcare appointment no-shows and building a predictive and visual Power BI dashboard. It aims to help healthcare providers minimize no-shows by identifying trends and influencing factors.

Files Included
- healthcare_noshows.csv - Dataset used
- healthcare_noshows_cleaned.xlsx: Cleaned dataset after removing unnecessary columns.
- Project 11: Power BI Dashboard
- Created and Tested model.ipynb (Python code): Model predicting patient no-shows.
- Healthcare_No_Show_Dashboard_Report.pdf: Formal 2-page project report.

Tools and Technologies Used
- Power BI: Data visualization and dashboard building.
- Python (pandas, numpy, scikit-learn): Data cleaning and model building.
- DAX: Custom KPIs and metrics inside Power BI.

Steps Completed
1. Data Cleaning
   - Cleaned dataset using Python libraries.
   - Dropped irrelevant columns and calculated wait times.

2. Data Analysis
   - Explored no-show trends based on SMS reminders, age, gender, and weekdays.

3. Model Building
   - Trained Decision Tree Classifier using scikit-learn.

4. Dashboard Development
   - Created multi-page dashboard with KPIs and visuals.
   - Applied dark theme and custom aesthetics.

5. Report Generation
   - Summarized project steps and outcomes in a formal 2-page report.

Conclusion
The project provides actionable insights into healthcare appointment no-shows and offers predictive capabilities. The dashboard enables healthcare administrators to identify trends and optimize appointment management strategies.
